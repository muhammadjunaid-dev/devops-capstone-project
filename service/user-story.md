# User Story Template

## Story Title
As a <type of user>, I want <some goal> so that <some reason>.

---

## Description
Describe the purpose of this user story and what functionality it should provide.

---

## Acceptance Criteria
- [ ] Condition 1 is satisfied  
- [ ] Condition 2 is satisfied  
- [ ] Condition 3 is satisfied  

---

## Definition of Done
- Code is implemented  
- Code is tested  
- Code is reviewed  
- Documentation is updated  
- No linting errors  
- All CI checks are passed  

---

## Tasks
1. Identify what needs to be done  
2. Implement the required code  
3. Write unit tests  
4. Update documentation  
5. Commit and push changes
